LEGO component for 3D printing. Design and 3D print your own LEGO components using this small tool — LEGO customised components developed through Python script using Rhino 3D software.

**How it works:**

1. Replace any surface with a LEGO unit.
2. Trim a hole.
3. Extrude surfaces.
4. Join all faces.
