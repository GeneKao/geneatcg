* GENEATCG — Personal Website Source

Source code for [[https://www.geneatcg.com][geneatcg.com]], the personal portfolio and blog of Gene Ting-Chun Kao.

Built with [[https://gohugo.io/][Hugo]] and the [[https://github.com/adityatelange/hugo-PaperMod][PaperMod]] theme with custom homepage layout.

* Setup

After cloning, initialize the theme submodule:

#+BEGIN_SRC shell
git submodule update --init
#+END_SRC

Install Hugo (macOS):

#+BEGIN_SRC shell
brew install hugo
#+END_SRC

* Run the dev server

#+BEGIN_SRC shell
hugo server --port 1313
#+END_SRC

Access at http://localhost:1313. Always use the dev server for previewing — do not open the =public/= folder directly in a browser.

* Deploy

Deployment is automatic: pushing to =main= triggers a GitHub Actions workflow that builds and deploys to [[https://github.com/GeneKao/genekao.github.io][genekao.github.io]].

For a manual build:

#+BEGIN_SRC shell
hugo --minify
cp CNAME public/
#+END_SRC

* Project structure

- =content/= — Site content in =.md= files. Key sections: =blog/=, =publications/=, =resume/=, =talks/=.
- =data/= — Structured YAML data for resume (timeline), publications (cards), and talks (cards).
- =layouts/= — Custom templates: =index.html= (homepage), =resume/=, =publications/=, =talks/=.
- =assets/css/extended/custom.css= — All custom styles and accent color overrides.
- =themes/PaperMod/= — Git submodule for the PaperMod theme.
- =static/= — Static assets (images, favicon).
- =.github/workflows/deploy.yml= — CI/CD: auto-builds and deploys on push to =main=.

* Adding a new blog post

#+BEGIN_SRC shell
hugo new content/blog/my-post-title/index.md
#+END_SRC

Then edit =content/blog/my-post-title/index.md=. Images can be placed in the same directory and referenced relatively.
