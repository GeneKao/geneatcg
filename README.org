* The source code of my website

Work in progress.

My website [[https://www.geneatcg.com][GENEATCG]] uses [[https://www.getlektor.com/][Lektor]] and [[https://github.com/spyder-ide/lektor-icon][lektor-icon theme]].


* Run the blog

#+BEGIN_SRC shell
/Users/kaot/Documents/GENEATCG/.venv/bin/lektor server --port 8000
#+END_SRC

Note: Port 5000 is used by a system process, so use 8000. Access at http://127.0.0.1:8000.

* Build website source 

#+BEGIN_SRC shell
/Users/kaot/Documents/GENEATCG/.venv/bin/lektor build --output-path ../genekao.github.io # build website source to directory
cp CNAME ../genekao.github.io/  # restore CNAME for custom domain
#+END_SRC
