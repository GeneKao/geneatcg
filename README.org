* GENEATCG — Personal Website Source

Source code for [[https://www.geneatcg.com][geneatcg.com]], the personal portfolio and blog of Gene Ting-Chun Kao.

Built with [[https://www.getlektor.com/][Lektor]] (Python static site generator) and the [[https://github.com/spyder-ide/lektor-icon][lektor-icon]] theme.

* Setup

After cloning, initialize the theme submodule:

#+BEGIN_SRC shell
git submodule update --init
#+END_SRC

Create and activate a virtual environment, then install Lektor and plugins:

#+BEGIN_SRC shell
python3 -m venv .venv
.venv/bin/pip install lektor lektor-google-analytics lektor-atom
#+END_SRC

* Run the dev server

#+BEGIN_SRC shell
.venv/bin/lektor server --port 8000
#+END_SRC

Note: Port 5000 is used by a macOS system process, so use 8000. Access at http://127.0.0.1:8000.

* Deploy

Deployment is automatic: pushing to =main= triggers a GitHub Actions workflow that builds and deploys to [[https://github.com/GeneKao/genekao.github.io][genekao.github.io]].

For manual builds:

#+BEGIN_SRC shell
.venv/bin/lektor build --output-path ../genekao.github.io --no-prune
cp CNAME ../genekao.github.io/  # restore CNAME for custom domain
#+END_SRC

* Project structure

- =content/= — Site content in =.lr= files. Key sections: =blog/=, =publications/=, =talks/=, =resume/=.
- =themes/lektor-icon/= — Git submodule for the theme (Jinja2 templates + assets).
- =assets/= — Overlaid on theme output. Contains =favicon.ico= and =static/css/custom-styles.css=.
- =geneatcg.lektorproject= — Project config: theme, plugins, deployment, and theme settings.
- =configs/google-analytics.ini= — Plugin config for =lektor-google-analytics=.
- =configs/atom.ini= — Plugin config for =lektor-atom= (RSS feed at =/blog/feed.xml=).
- =.github/workflows/deploy.yml= — CI/CD: auto-builds and deploys on push to =main=.
